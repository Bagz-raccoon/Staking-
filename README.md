## Deploy to Vercel via GitHub
1. Upload the contents of this folder to your GitHub repo (root is fine).
2. Vercel will auto-deploy.
3. Your portal will be available at:
   - https://YOUR-VERCEL-DOMAIN/ (or /index.html)

## Customize
- Replace ./assets/bagz-header.svg with your official header graphic (keep filename).
- Edit tiers and APY text inside index.html.
